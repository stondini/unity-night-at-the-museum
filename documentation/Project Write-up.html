Introduction

Night at the Museum is a project of Udacity VR Developer nano-degree (term 2).
The museum visitor can walk to reach the viewing stations and activate the station's related content.
This write up presents the project achievement process that starts with Persona description, scene sketches & ideas, sound amp; light environment, user testing and deployment on a device.
The entire process was executed iteratively (agile methodology).

Goals of this write up :

  • Tests feedback. They are useful to improve the experience.

    Sketche and Ideas

    The idea is to build a museum that gives informations on some famous Swiss Alps mountains.
    The visitor can move around mountains using a way points navigation system.
    Each mountain is identified by a pinpoint which is displaying the name and altitude of the related mountain.


    Swiss Alps

    The user can visit the must fabulous Swiss Alps Mountains my moving along a path (red line below).

    Design

    The first think to do is to build the mesh of the Swiss Alps.
    Swiss Government provides a 3D points cloud of the whole country (Swiss Topo).
    The points cloud must be converted to a mesh (I tried few apps like CloudCompare, MeshLab and Blender. I finally used Blender). The useless points outside the Alps area are removed.

    The second step is to map the texture on the mesh (This took me a lot of time because it was the first time I used Blender app).

    Next, the mesh and its texture are exported from Blender and imported into Unity.
    Pinpoints are placed on the top of interesting mountains (I used Google Map to locate the position of the mountains as precisely as possible.).

    Wayspoints are positionned to allow the visitor reaching all interesting mountains.

    Last step is to code all the glue :

    • Application controller that initialize the pinpoints and the camera.
    • Waypoints navigation. This could be the same code as A Maze project.
    • UI interactions.
    • UI WebView to display the Wikipedia page of the selected mountains.

    Scene

    Each important mountain is pointed by a pinpoint.
  • A pinpoint shows a mountain name and altitude.
  • The user can click on a pinpoint to display the Wikipedia description on a giant virtual screen.
  • The pinpoints must be always visible to the user. It means, the pinpoints must look at or facing the user/camera at any time.
  • The cloudy skybox improve the scene's reality.
  • The sound of a windy weather improve the scene's reality
  • The UI canvas contains few toggle buttons to allow way point altitude change.

    Next Steps

    I'd like to build a more precise mesh. More points.
    The purpose is to provide many more mountains with a very high resolution texture. Maybe many textures should be used. One by mesh part.

    Visitor experience could be improved by letting it navigate freely.
    Waypoints will be removed and replaced by a navigation system that requires specific hardware like hand controllers.

    The WebView is not good. Maybe I should consider a commercial WebView. I could also check if the last Unity version provides a WebView.
